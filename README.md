# Hostel Rent Calculator
## Description
​This project is a simple Python tool that helps roommates calculate their total hostel expenses, including food and electricity, and divide the total amount among themselves.
## ​Features
* ​Calculate Rent: Total room rent and food charges.
​* Electricity Splitter: Calculates bill based on units and rate.
* ​Equal Division: Automatically splits the grand total by the number of people.
## ​How to Run
* ​Download or clone this repository.
* ​Open your terminal or command prompt.
* ​Run the script using: python Rent_Calculator.py
